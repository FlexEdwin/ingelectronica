Ejercicio 11: Ecuación Diferencial Exacta (Directa)

$$(2xy^3 + 5)dx + (3x^2y^2 + 4y)dy = 0$$

Clasificación Obligatoria:
Orden: Primer orden.
Linealidad: No lineal.
Homogeneidad: Sí es Homogénea ($=0$).

Paso 0: Prueba de Exactitud

$M(x,y) = 2xy^3 + 5$

$N(x,y) = 3x^2y^2 + 4y$

$$\frac{\partial M}{\partial y} = 2x(3y^2) + 0 = 6xy^2$$
$$\frac{\partial N}{\partial x} = 3y^2(2x) + 0 = 6xy^2$$

Como $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$, Es una ED exacta.

Paso 1: Integración
$$f(x,y) = \int (2xy^3 + 5) dx = 2y^3 \left( \frac{x^2}{2} \right) + 5x + g(y)$$
$$f(x,y) = x^2y^3 + 5x + g(y)$$
Paso 2: Derivación parcial e igualación
$$\frac{\partial f}{\partial y} = x^2(3y^2) + 0 + g'(y) = 3x^2y^2 + g'(y)$$

Igualamos con $N(x,y)$:

$$3x^2y^2 + g'(y) = 3x^2y^2 + 4y \implies g'(y) = 4y$$
Paso 3 y 4: Hallar $g(y)$ y Solución
$$g(y) = \int 4y dy = 2y^2$$

Solución:

$$x^2y^3 + 5x + 2y^2 = C$$

Ejercicio 12: Transformable a Exacta (Con Factor Integrante)

$$(3xy^2 + 2y)dx + (2x^2y + x)dy = 0$$

Clasificación Obligatoria:
Orden: Primer orden.
Linealidad: No lineal.
Homogeneidad: Sí es Homogénea ($=0$).

Ensayo y Error (Prueba de Exactitud inicial)

$M = 3xy^2 + 2y \implies \frac{\partial M}{\partial y} = 6xy + 2$

$N = 2x^2y + x \implies \frac{\partial N}{\partial x} = 4xy + 1$

No es exacta.

Búsqueda del Factor Integrante
$$P(x) = \frac{\frac{\partial M}{\partial y} - \frac{\partial N}{\partial x}}{N} = \frac{(6xy + 2) - (4xy + 1)}{2x^2y + x} = \frac{2xy + 1}{x(2xy + 1)} = \frac{1}{x}$$
$$I(x) = e^{\int \frac{1}{x} dx} = e^{\ln x} = x$$
Transformación y Nueva Prueba
$$(3x^2y^2 + 2xy)dx + (2x^3y + x^2)dy = 0$$
$$\frac{\partial M_{nuevo}}{\partial y} = 6x^2y + 2x \quad \text{y} \quad \frac{\partial N_{nuevo}}{\partial x} = 6x^2y + 2x$$

Es exacta.

Pasos de Solución
$$f(x,y) = \int (2x^3y + x^2) dy = x^3y^2 + x^2y + g(x)$$
$$\frac{\partial f}{\partial x} = 3x^2y^2 + 2xy + g'(x) = 3x^2y^2 + 2xy \implies g'(x) = 0 \implies g(x) = 0$$

Solución:

$$x^3y^2 + x^2y = C$$

Ejercicio 13: Ecuación Diferencial Exacta (Directa)

$$(4x^3y^3 + 2y^2)dx + (3x^4y^2 + 4xy)dy = 0$$

Clasificación Obligatoria:
Orden: Primer orden.
Linealidad: No lineal.
Homogeneidad: Sí es Homogénea ($=0$).

Paso 0: Prueba de Exactitud
$$\frac{\partial M}{\partial y} = 4x^3(3y^2) + 2(2y) = 12x^3y^2 + 4y$$
$$\frac{\partial N}{\partial x} = 3y^2(4x^3) + 4y(1) = 12x^3y^2 + 4y$$

Es una ED exacta.

Paso 1: Integración
$$f(x,y) = \int (4x^3y^3 + 2y^2) dx = x^4y^3 + 2xy^2 + g(y)$$
Paso 2, 3 y 4: Derivación, Igualación y Solución
$$\frac{\partial f}{\partial y} = 3x^4y^2 + 4xy + g'(y) = 3x^4y^2 + 4xy \implies g'(y) = 0 \implies g(y) = 0$$

Solución:

$$x^4y^3 + 2xy^2 = C$$

Ejercicio 14: Transformable a Exacta (Con Factor Integrante)

$$(2xy^2 + 3y)dx + (x^2y + x)dy = 0$$

Clasificación Obligatoria:
Orden: Primer orden.
Linealidad: No lineal.
Homogeneidad: Sí es Homogénea ($=0$).

Ensayo y Error (Prueba de Exactitud inicial)
$$\frac{\partial M}{\partial y} = 4xy + 3 \quad \text{y} \quad \frac{\partial N}{\partial x} = 2xy + 1$$

No es exacta.

Búsqueda del Factor Integrante
$$P(x) = \frac{(4xy + 3) - (2xy + 1)}{x^2y + x} = \frac{2xy + 2}{x(xy + 1)} = \frac{2(xy + 1)}{x(xy + 1)} = \frac{2}{x}$$
$$I(x) = e^{\int \frac{2}{x} dx} = e^{2 \ln x} = x^2$$
Transformación y Nueva Prueba
$$(2x^3y^2 + 3x^2y)dx + (x^4y + x^3)dy = 0$$
$$\frac{\partial M_{nuevo}}{\partial y} = 4x^3y + 3x^2 \quad \text{y} \quad \frac{\partial N_{nuevo}}{\partial x} = 4x^3y + 3x^2$$

Es exacta.

Pasos de Solución
$$f(x,y) = \int (x^4y + x^3) dy = \frac{x^4y^2}{2} + x^3y + g(x)$$
$$\frac{\partial f}{\partial x} = 2x^3y^2 + 3x^2y + g'(x) = 2x^3y^2 + 3x^2y \implies g'(x) = 0 \implies g(x) = 0$$

Multiplicamos todo por 2 para limpiar denominadores:
Solución:

$$x^4y^2 + 2x^3y = K$$

Ejercicio 15: Ecuación Diferencial Exacta (Directa)

$$(4xy^3 + 3y^2 + 3x^2)dx + (6x^2y^2 + 6xy)dy = 0$$

Clasificación Obligatoria:
Orden: Primer orden.
Linealidad: No lineal.
Homogeneidad: Sí es Homogénea ($=0$).

Paso 0: Prueba de Exactitud
$$\frac{\partial M}{\partial y} = 12xy^2 + 6y \quad \text{y} \quad \frac{\partial N}{\partial x} = 12xy^2 + 6y$$

Es una ED exacta.

Paso 1: Integración
$$f(x,y) = \int (4xy^3 + 3y^2 + 3x^2) dx = 2x^2y^3 + 3xy^2 + x^3 + g(y)$$
Paso 2, 3 y 4: Derivación, Igualación y Solución
$$\frac{\partial f}{\partial y} = 6x^2y^2 + 6xy + g'(y) = 6x^2y^2 + 6xy \implies g'(y) = 0 \implies g(y) = 0$$

Solución:

$$2x^2y^3 + 3xy^2 + x^3 = C$$

Ejercicio 16: Transformable a Exacta (Con Factor Integrante)

$$(2xy^2 + y)dx + (3x^2y + 2x)dy = 0$$

Clasificación Obligatoria:
Orden: Primer orden.
Linealidad: No lineal.
Homogeneidad: Sí es Homogénea ($=0$).

Ensayo y Error (Prueba de Exactitud inicial)
$$\frac{\partial M}{\partial y} = 4xy + 1 \quad \text{y} \quad \frac{\partial N}{\partial x} = 6xy + 2$$

No es exacta.

Búsqueda del Factor Integrante (Dependiente de y)
$$P(y) = \frac{\frac{\partial N}{\partial x} - \frac{\partial M}{\partial y}}{M} = \frac{(6xy + 2) - (4xy + 1)}{2xy^2 + y} = \frac{2xy + 1}{y(2xy + 1)} = \frac{1}{y}$$
$$I(y) = e^{\int \frac{1}{y} dy} = e^{\ln y} = y$$
Transformación y Nueva Prueba
$$(2xy^3 + y^2)dx + (3x^2y^2 + 2xy)dy = 0$$
$$\frac{\partial M_{nuevo}}{\partial y} = 6xy^2 + 2y \quad \text{y} \quad \frac{\partial N_{nuevo}}{\partial x} = 6xy^2 + 2y$$

Es exacta.

Pasos de Solución
$$f(x,y) = \int (2xy^3 + y^2) dx = x^2y^3 + xy^2 + g(y)$$
$$\frac{\partial f}{\partial y} = 3x^2y^2 + 2xy + g'(y) = 3x^2y^2 + 2xy \implies g'(y) = 0 \implies g(y) = 0$$

Solución:

$$x^2y^3 + xy^2 = C$$

Ejercicio 17: Ecuación Diferencial Exacta (Directa)

$$(2xy^4 + 3x^2)dx + (4x^2y^3 + 2y)dy = 0$$

Clasificación Obligatoria:
Orden: Primer orden.
Linealidad: No lineal.
Homogeneidad: Sí es Homogénea ($=0$).

Paso 0: Prueba de Exactitud
$$\frac{\partial M}{\partial y} = 8xy^3 \quad \text{y} \quad \frac{\partial N}{\partial x} = 8xy^3$$

Es una ED exacta.

Paso 1: Integración
$$f(x,y) = \int (2xy^4 + 3x^2) dx = x^2y^4 + x^3 + g(y)$$
Paso 2, 3 y 4: Derivación, Igualación y Solución
$$\frac{\partial f}{\partial y} = 4x^2y^3 + g'(y) = 4x^2y^3 + 2y \implies g'(y) = 2y \implies g(y) = y^2$$

Solución:

$$x^2y^4 + x^3 + y^2 = C$$

Ejercicio 18: Transformable a Exacta (Con Factor Integrante)

$$(4x^2y + 2y^2)dx + (x^3 + 2xy)dy = 0$$

Clasificación Obligatoria:
Orden: Primer orden.
Linealidad: No lineal.
Homogeneidad: Sí es Homogénea ($=0$).

Ensayo y Error (Prueba de Exactitud inicial)
$$\frac{\partial M}{\partial y} = 4x^2 + 4y \quad \text{y} \quad \frac{\partial N}{\partial x} = 3x^2 + 2y$$

No es exacta.

Búsqueda del Factor Integrante
$$P(x) = \frac{(4x^2 + 4y) - (3x^2 + 2y)}{x^3 + 2xy} = \frac{x^2 + 2y}{x(x^2 + 2y)} = \frac{1}{x}$$
$$I(x) = e^{\int \frac{1}{x} dx} = e^{\ln x} = x$$
Transformación y Nueva Prueba
$$(4x^3y + 2xy^2)dx + (x^4 + 2x^2y)dy = 0$$
$$\frac{\partial M_{nuevo}}{\partial y} = 4x^3 + 4xy \quad \text{y} \quad \frac{\partial N_{nuevo}}{\partial x} = 4x^3 + 4xy$$

Es exacta.

Pasos de Solución
$$f(x,y) = \int (x^4 + 2x^2y) dy = x^4y + x^2y^2 + g(x)$$
$$\frac{\partial f}{\partial x} = 4x^3y + 2xy^2 + g'(x) = 4x^3y + 2xy^2 \implies g'(x) = 0 \implies g(x) = 0$$

Solución:

$$x^4y + x^2y^2 = C$$

Ejercicio 19: Ecuación Diferencial Exacta (Directa)

$$(3x^2y^4 + 2xy)dx + (4x^3y^3 + x^2)dy = 0$$

Clasificación Obligatoria:
Orden: Primer orden.
Linealidad: No lineal.
Homogeneidad: Sí es Homogénea ($=0$).

Paso 0: Prueba de Exactitud
$$\frac{\partial M}{\partial y} = 12x^2y^3 + 2x \quad \text{y} \quad \frac{\partial N}{\partial x} = 12x^2y^3 + 2x$$

Es una ED exacta.

Paso 1: Integración
$$f(x,y) = \int (3x^2y^4 + 2xy) dx = x^3y^4 + x^2y + g(y)$$
Paso 2, 3 y 4: Derivación, Igualación y Solución
$$\frac{\partial f}{\partial y} = 4x^3y^3 + x^2 + g'(y) = 4x^3y^3 + x^2 \implies g'(y) = 0 \implies g(y) = 0$$

Solución:

$$x^3y^4 + x^2y = C$$

Ejercicio 20: Transformable a Exacta (Con Factor Integrante)

$$(3x^2y^2 + 5xy)dx + (x^3y + x^2)dy = 0$$

Clasificación Obligatoria:
Orden: Primer orden.
Linealidad: No lineal.
Homogeneidad: Sí es Homogénea ($=0$).

Ensayo y Error (Prueba de Exactitud inicial)
$$\frac{\partial M}{\partial y} = 6x^2y + 5x \quad \text{y} \quad \frac{\partial N}{\partial x} = 3x^2y + 2x$$

No es exacta.

Búsqueda del Factor Integrante
$$P(x) = \frac{(6x^2y + 5x) - (3x^2y + 2x)}{x^3y + x^2} = \frac{3x^2y + 3x}{x(x^2y + x)} = \frac{3x(xy + 1)}{x^2(xy + 1)} = \frac{3}{x}$$
$$I(x) = e^{\int \frac{3}{x} dx} = e^{3 \ln x} = x^3$$
Transformación y Nueva Prueba
$$(3x^5y^2 + 5x^4y)dx + (x^6y + x^5)dy = 0$$
$$\frac{\partial M_{nuevo}}{\partial y} = 6x^5y + 5x^4 \quad \text{y} \quad \frac{\partial N_{nuevo}}{\partial x} = 6x^5y + 5x^4$$

Es exacta.

Pasos de Solución
$$f(x,y) = \int (x^6y + x^5) dy = \frac{x^6y^2}{2} + x^5y + g(x)$$
$$\frac{\partial f}{\partial x} = 3x^5y^2 + 5x^4y + g'(x) = 3x^5y^2 + 5x^4y \implies g'(x) = 0 \implies g(x) = 0$$

Multiplicamos por 2 para eliminar la fracción:
Solución:

$$x^6y^2 + 2x^5y = K$$
